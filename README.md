# Ponuda → Troškovnik

Pretvara Viessmann PDF ponudu u Excel troškovnik.

## Korištenje

1. Uploadaj PDF ponudu.
2. Provjeri prepoznate stavke i kontrolni zbroj.
3. Klikni **Preuzmi Excel**.

Podržane su HR ponude (`Kom`, `Poz.`) i EN ponude (`PC`, `Pos.`).

## Zahtjevi

- PDF mora imati tekstualni sloj (ne sken).
